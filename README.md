# Next.js Hello World

This is a simple Next.js project initialized with JavaScript, ESLint, Tailwind CSS, and App Router.

## Getting Started

1. Install dependencies (if not already):
   ```bash
   npm install
   ```
2. Run the development server:
   ```bash
   npm run dev
   ```
3. Open [http://localhost:3000](http://localhost:3000) to view your app.

## Features
- Next.js (JavaScript)
- ESLint
- Tailwind CSS
- App Router

## Customization
Edit the files in the `app/` directory to update your Hello World page.

This project uses [`next/font`](https://nextjs.org/docs/app/building-your-application/optimizing/fonts) to automatically optimize and load [Geist](https://vercel.com/font), a new font family for Vercel.
